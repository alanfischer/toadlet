*** The Toadlet Engine depreciated folder ***

The files in this depreciated folder are not part
of the standard toadlet build system. They contain 
old code and functionality that is out of date or
has been removed from the engine. The files are kept
here as a reference.

None of these files should be expected to build or be
functional.

Specifically, the old J2ME builds of toadlet may be
found here, along with the build system code.
